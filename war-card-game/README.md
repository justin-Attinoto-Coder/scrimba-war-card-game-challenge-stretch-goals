# War Card Game

Welcome to the War Card Game web application! This project is built using Next.js and TypeScript, featuring advanced animations and gradient effects to create a visually stunning gaming experience.

- [Features](#features)

## Features

- Engaging War card game mechanics
- Beautiful animations for card flipping and shuffling
- Gradient effects for a modern look
- Responsive design for mobile and desktop devices
- API integration for game state management

## Installation

To get started with the War Card Game, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/war-card-game.git
   ```

2. Navigate to the project directory:

   ```bash
   cd war-card-game
   ```

3. Install the dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```

5. Open your browser and visit `http://localhost:3000` to see the application in action!

## Usage

Once the application is running, you can start a new game by clicking the "Start Game" button on the main page. The game will manage player turns and display scores in real-time.
